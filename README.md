# Simple NixOS helper using Go & GTK3
### Provided:
- Search on
	- https://search.nixos.org/packages
	- https://nixos.wiki/
- gedit to open configuration.nix, {packages, specialisation}.nix
- test current configuration
- channels update
- rebuild to boot
- garbage collect
- store optimization
- simple logging
